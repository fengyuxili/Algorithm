# 快速排序
* 最流行速度最快的高级排序算法

* 通过一趟排序将要排序的数据分割成独立的两部分，其中一部分的所有数据都比另外一部分的所有数据都要小，<br>
然后再按此方法对这两部分数据分别进行快速排序，整个排序过程可以递归进行，以此达到整个数据变成有序序列。<br>
第一个选为基准 小的放左边大的放右边<br>

算法描述：

下标索引left指向第一个 right指向最后一个<br>
基准=第一个或最后一个
i=left,j=right<br>
i++ 直到>=基准 ; j-- 直到<=基准<br>
若i<j i与j 指向的值交换,不断循环;小于基准的放左边,大于基准的放右边<br>
直到i>=j 基准与j指向的值交换<br>

递归:<br>
左子序列 
基准等于第一个数或j-1的数
i=left j=j-1
...

右子序列
基准等于j+1个数或最后一个数
i=j+1 j=right
...
