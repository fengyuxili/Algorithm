# 递归算法
* 递归是自己调用自己
* 递归需要建立函数副本缓存需要消耗内存缓存 数据多运行就会变慢，迭代【循环】不需要缓存

## 递归的基本准则
* 基准情形 需要不用递归就能求得的解
* 不断推进 递归不断向基准推进
* 设计法则 所有的递归都能运行
* 合成效益法则

## 折半查找 递归
* 前提是排好序的序列
* 函数形参加上left和right
* 通过判断位于左边一半或右边一半不断的折半查找
* 通过更新x > list[mid] , left=mid+1; 或者x < list[mid] , right=mid-1;序列起点或终点
* 通过left > right 跳出迭代
